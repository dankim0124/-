#contract
## 내용
 - 네이버 뉴스 URL 을 입력받아 같은 카테고리의 뉴스 중 유사도가 가장 높은 뉴스 5개를 추천해 주는 웹 사이트 입니다.
 
 - 가져오는 뉴스는 카테고리별 조회수가 높은 100개의 뉴스 중에서 유사도를 검사합니다  
 - 유사도 검사는 코사인 거리를 이용해 결과를 리턴하고, 자주 조회될 것으로 예상되는 정치 뉴스는 TF-IDF 결과값을 미리 계산해 두었다가 요청이 들어오면 바로 리턴해 줍니다.
   ㄴ> 코사인 거리만을 이용한 방법은 뉴스 안에서의 단어의 희귀도를 반영하지 못하고, Tf-IDF 를 계산하는 방법은 연산이 시간이오래 걸립니다.
        즉, 빠르고 덜 정확한 방법을 일반적으로 사용하고, 요청이 많이 올것으로 예산되는 범위에 한에서는 결과를 미리 저장해두었다 요청이 들어오면 바로 리턴해 줍니다.
 
 
 ## 프로세스
    1. 서버가 시작되면 우선적으로 당일의 랭킹 뉴스 페이지의 html의 body에서 li.ranking_list를 검사하여 최신 뉴스의 url을 가져옵니다.
    2. 1에서 찾은 각 뉴스들을 각 단어 수와 TF-IDF 가중치를 기록한 객체를 생성합니다.
